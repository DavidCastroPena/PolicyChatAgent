# Pipeline Reports

This folder contains all generated reports from the PolicyChatAgent pipeline runs.

## Folder Structure

- **combined/**: Combined reports merging local and external paper analysis
- **external_papers/**: Reports from Semantic Scholar API searches
- **local_papers/**: Reports from local paper repository analysis
- **questions/**: Generated questions and comparison questions
- **memos/**: Final policy memos generated by the pipeline

## File Naming Convention

All files are timestamped with the format: `YYYYMMDD_HHMMSS`

Example: `combined_report_20260118_215017.jsonl`

## Note

This entire `reports/` folder is gitignored to keep the repository clean. Reports are generated locally during pipeline execution.
